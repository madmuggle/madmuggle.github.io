# Moana: How Far I'll Go
2016/12/10 22:11:00
Misc


迪士尼的又一部好作品。这部电影肯定不可能像「Zootopia」那样，对上所有人的口味，但是它会令很多人产生强烈的共鸣。

最后打动我们的，终归只是简单的情感，而非所谓的精彩曲折的故事。好多电影走错了方向，好多人也是。
